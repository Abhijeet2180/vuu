Vuu - The Realtime View Server

Wire Protocol

The wire protocol is JSON based. It requires a user to authenticate against the server (Athentication provider is pluggable) When a user authenticates, a conversation is undertable to send down meta to the client relating to the tables and fields available.

Then in general a view port is requested, the server responds as success or failure and then starts streaming data.
